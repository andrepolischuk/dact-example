# recd-example

> An example of recd application

## Install

```sh
git clone https://github.com/andrepolischuk/recd-example
cd recd-example
npm install
```

## Run

```sh
npm start
```

## Test

```sh
npm test
```

## Build

```sh
npm run build
```
