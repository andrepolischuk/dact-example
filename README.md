# dact-example

> An example of dact application

## Install

```sh
git clone https://github.com/andrepolischuk/dact-example
cd dact-example
npm install
```

## Run

```sh
npm start
```

## Test

```sh
npm test
```

## Build

```sh
npm run build
```
